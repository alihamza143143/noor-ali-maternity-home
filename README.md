# Noor Ali Clinic & Maternity Home — Website

A fast, mobile-first single-page website for Noor Ali Clinic & Maternity Home,
Sardar Market, Kallar Syedan Road.

## View it

Just double-click **`index.html`** — it opens in any browser. No installation needed.

## Files

| File | What it is |
|------|-----------|
| `index.html` | The whole website (all content + the mother & child logo) |
| `style.css` | Colours, layout, responsive design (Warm & Nurturing theme) |
| `script.js` | Mobile menu, smooth scrolling, scroll animations |
| `README.md` | This file |

- **Address** — search `Sardar Market`.
- **Services** — edit the `service-card` blocks in the Services section.
- **Colours** — change the variables at the top of `style.css` (`:root { ... }`).

## Putting it online (free options)

1. **Netlify Drop** — go to app.netlify.com/drop and drag this folder in. Done.
2. **Vercel** — vercel.com → import the folder.
3. **GitHub Pages** — push this folder to a repo and enable Pages.

All three give a free public link you can share on Google, Facebook, etc.

## Notes

- Contact works through one-tap **Call** and **WhatsApp** buttons — no backend needed.
- The Google Map is a free embed pointing at "Sardar Market, Kallar Syedan Road".
  To pin the exact spot, open the location in Google Maps, copy the embed code,
  and replace the `<iframe>` in the **Visit Us** section.
